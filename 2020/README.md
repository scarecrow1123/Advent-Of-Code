### AOC 2020

#### Build Instructions

```bash
$ git clone https://github.com/scarecrow1123/Advent-Of-Code
$ cd Advent-Of-Code/2020
$ mkdir build && cd build
$ cmake <path to src> .
$ cmake --build .. --target aoc2020_day<number>
# <number> runs from 1 to 25. ex: day1's target is `aoc2020_day1`
$ ./aoc2020_day<number>
```
